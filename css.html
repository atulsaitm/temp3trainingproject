Advanced CSS Styling

Master modern CSS techniques including flexbox, grid, animations, and advanced selectors. Learn how to create beautiful, responsive layouts.

CSS Layout Techniques

Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

.item {
    flex: 1;
    min-width: 200px;
}

CSS Grid

Grid is a two-dimensional layout method for creating complex layouts with rows and columns.

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}

.grid-item {
    background: #f0f0f0;
    padding: 20px;
}

CSS Animations & Transitions

Transitions

Smooth property changes from one value to another.

.button {
    background: #007bff;
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.button:hover {
    background: #0056b3;
    transform: translateY(-2px);
}

Keyframes Animation

Create complex animations with multiple steps.

@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.element {
    animation: fadeIn 0.5s ease-out;
}

Advanced Selectors

  • Attribute Selectors
    input[type="text"] { border: 1px solid #ccc; }
  • Pseudo-classes
    :hover, :focus, :nth-child(even)
  • Pseudo-elements
    ::before, ::after, ::first-line
  • Combinators
    Descendant (space), Child (>), Adjacent (+), General Sibling (~)

Modern CSS Features

CSS Custom Properties (Variables)

:root {
    --primary-color: #007bff;
    --font-size: 16px;
    --spacing: 20px;
}

.button {
    background: var(--primary-color);
    font-size: var(--font-size);
    padding: var(--spacing);
}

CSS Functions

  • calc()
    width: calc(100% - 20px);
  • min(), max(), clamp()
    font-size: clamp(1rem, 2vw, 2rem);
  • rgb(), hsl()
    background: hsl(220, 100%, 50%);
  • linear-gradient(), radial-gradient()
    background: linear-gradient(45deg, #ff0000, #0000ff);

Responsive Design

Use media queries to create responsive layouts that adapt to different screen sizes.

/* Mobile first */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 992px) {
    .container {
        width: 970px;
    }
}

CSS Best Practices

  • Use semantic class names and avoid presentational names
  • Organize CSS with a logical structure
  • Use CSS variables for maintainable code
  • Minimize specificity conflicts
  • Test across different browsers and devices
  • Use modern CSS features with fallbacks
