JavaScript Fundamentals

Learn the core concepts of JavaScript programming. From variables and functions to DOM manipulation and event handling.

Getting Started with JavaScript

JavaScript is a programming language that adds interactivity to web pages. It can be added to HTML in three ways:

  • Internal JavaScript
    <script> tags in HTML
  • External JavaScript
    <script src="script.js"></script>
  • Inline JavaScript
    Event handlers in HTML attributes

Variables and Data Types

// Variable declarations
let name = "John";           // String
const age = 25;              // Number
var isStudent = true;        // Boolean
let scores = [85, 92, 78];   // Array
let person = {               // Object
    name: "John",
    age: 25,
    isStudent: true
};

Functions

Functions are reusable blocks of code that perform specific tasks.

// Function declaration
function greetUser(name) {
    return `Hello, ${name}!`;
}

// Function expression
const calculateArea = function(width, height) {
    return width * height;
};

// Arrow function
const multiply = (a, b) => a * b;

// Function call
console.log(greetUser("Alice"));

Control Flow

Conditional Statements

let temperature = 25;

if (temperature > 30) {
    console.log("It's hot!");
} else if (temperature > 20) {
    console.log("It's warm.");
} else {
    console.log("It's cool.");
}

// Ternary operator
let message = temperature > 20 ? "Warm" : "Cool";

Loops

// For loop
for (let i = 0; i < 5; i++) {
    console.log(`Count: ${i}`);
}

// For...of loop (arrays)
let fruits = ["apple", "banana", "orange"];
for (let fruit of fruits) {
    console.log(fruit);
}

// While loop
let count = 0;
while (count < 3) {
    console.log(`While count: ${count}`);
    count++;
}

DOM Manipulation

The Document Object Model (DOM) represents the structure of HTML documents.

// Selecting elements
const heading = document.getElementById("myHeading");
const paragraphs = document.querySelectorAll("p");
const button = document.querySelector(".btn");

// Changing content
heading.textContent = "New Heading";
heading.innerHTML = "<strong>Bold Heading</strong>";

// Modifying styles
heading.style.color = "blue";
heading.style.fontSize = "24px";

// Adding/removing classes
heading.classList.add("highlight");
heading.classList.remove("old-class");
heading.classList.toggle("active");

Event Handling

Events allow JavaScript to respond to user interactions.

// Method 1: HTML attribute
// <button onclick="handleClick()">Click me</button>

// Method 2: Event listener
const button = document.querySelector("#myButton");

button.addEventListener("click", function() {
    console.log("Button clicked!");
});

// Method 3: Arrow function
button.addEventListener("click", () => {
    button.textContent = "Clicked!";
});

// Common events: click, mouseover, mouseout, keydown, submit

Arrays and Objects

Array Methods

let numbers = [1, 2, 3, 4, 5];

// Adding elements
numbers.push(6);        // [1, 2, 3, 4, 5, 6]
numbers.unshift(0);     // [0, 1, 2, 3, 4, 5, 6]

// Removing elements
numbers.pop();          // [0, 1, 2, 3, 4, 5]
numbers.shift();        // [1, 2, 3, 4, 5]

// Transforming arrays
let doubled = numbers.map(num => num * 2);     // [2, 4, 6, 8, 10]
let even = numbers.filter(num => num % 2 === 0); // [2, 4]
let sum = numbers.reduce((acc, num) => acc + num, 0); // 15

Working with Objects

let user = {
    name: "John",
    age: 30,
    email: "john@example.com"
};

// Accessing properties
console.log(user.name);        // "John"
console.log(user["age"]);      // 30

// Adding/modifying properties
user.city = "New York";
user.age = 31;

// Object methods
let keys = Object.keys(user);      // ["name", "age", "email", "city"]
let values = Object.values(user);  // ["John", 31, "john@example.com", "New York"]

// Destructuring
let { name, age } = user;
console.log(name, age);            // "John" 31

Asynchronous JavaScript

// Promises
function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Data loaded!");
        }, 1000);
    });
}

fetchData()
    .then(result => console.log(result))
    .catch(error => console.error(error));

// Async/await
async function loadData() {
    try {
        const result = await fetchData();
        console.log(result);
    } catch (error) {
        console.error(error);
    }
}

Best Practices

  • Use const by default, let when reassignment is needed
  • Use descriptive variable and function names
  • Comment your code for clarity
  • Handle errors appropriately
  • Use modern ES6+ features when possible
  • Test your code thoroughly
