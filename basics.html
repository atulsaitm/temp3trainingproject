HTML & CSS Basics

Learn the fundamental building blocks of modern web development. This guide covers everything you need to know to get started with HTML and CSS.

HTML Fundamentals

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of your website.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Web Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

Common HTML Elements

  • <h1> to <h6>
    Headings - h1 is the most important, h6 is the least
  • <p>
    Paragraphs of text
  • <a>
    Links to other pages or resources
  • <img>
    Images and graphics
  • <ul> and <ol>
    Unordered and ordered lists
  • <div>
    Generic container for grouping elements

CSS Fundamentals

CSS (Cascading Style Sheets) controls the appearance and layout of your HTML elements. It separates content from presentation.

Adding CSS to HTML

  • External CSS
    <link rel="stylesheet" href="style.css">
  • Internal CSS
    <style> in the <head> section
  • Inline CSS
    style attribute on HTML elements

CSS Syntax

selector {
    property: value;
    another-property: another-value;
}

Common CSS Properties

  • color
    Text color (e.g., color: #ff0000;)
  • font-size
    Text size (e.g., font-size: 16px;)
  • background-color
    Background color (e.g., background-color: #f0f0f0;)
  • width & height
    Element dimensions (e.g., width: 100px;)
  • margin & padding
    Spacing around and inside elements
  • border
    Element borders (e.g., border: 1px solid #000;)

The Box Model

Every HTML element is treated as a rectangular box. The box model consists of content, padding, border, and margin.

Box Model Diagram

Margin → Border → Padding → Content

Selectors

  • Element Selector
    p { color: blue; } - selects all <p> elements
  • Class Selector
    .highlight { background: yellow; } - selects elements with class="highlight"
  • ID Selector
    #header { font-size: 24px; } - selects element with id="header"
  • Universal Selector
    * { margin: 0; } - selects all elements
